Automatically add Facebook Open Graph and Twitter Card support to your WordPress website.

== Description ==

Add facebook and twitter sharing for your blog posts and pages!  Sharing your blog posts with social media sharing automatically provides important details like featured images, title, and description to your links so they look fantastic on facebook or twitter.

**Note:** Twitter Card support requires approval from twitter. You can request approval at the [twitter card validator](https://dev.twitter.com/docs/cards/validation/validator).

== Installation ==

1. Install Social Media Sharing via the WordPress.org plugin directory
2. Activate `Social Media Sharing` from the 'Plugins' menu in WordPress
3. Configure your default image, description and twitter account from the 'Social Media Sharing' settings menu under 'Settings'
